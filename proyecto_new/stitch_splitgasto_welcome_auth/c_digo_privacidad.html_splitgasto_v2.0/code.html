Privacidad SplitGasto

Privacidad SplitGasto v2.0

Última actualización: Mayo 2024.

SplitGasto se compromete a proteger tu privacidad y seguridad de datos. Esta política describe cómo gestionamos tus datos personales dentro de la aplicación para garantizar transparencia total.

1.

Información General

Nuestra aplicación está diseñada para facilitar la división de gastos entre amigos y grupos. Al utilizar SplitGasto, aceptas las prácticas descritas en este documento. Operamos bajo estándares de seguridad modernos para proteger tu información.

2.

Datos Recogidos

Recopilamos la información mínima necesaria para el funcionamiento:

  • Nombre de usuario y dirección de correo electrónico.
  • Nombres de grupos y descripciones de gastos.
  • Identificadores de dispositivo para notificaciones push.
3.

Uso de Información

Los datos se utilizan exclusivamente para sincronizar cuentas entre dispositivos, calcular deudas de forma precisa y enviar alertas sobre nuevos gastos o pagos pendientes. No vendemos tus datos a terceros.

4.

Cookies y Almacenamiento

Utilizamos LocalStorage y pequeñas cookies técnicas para mantener tu sesión iniciada y recordar tus preferencias de tema (Modo Oscuro/Claro). Puedes gestionar estas preferencias desde los ajustes de tu navegador.

5.

Derechos del Usuario

Tienes derecho a acceder, rectificar o eliminar tus datos en cualquier momento. Para ejercer estos derechos o solicitar el cierre permanente de tu cuenta, puedes contactarnos a través de nuestro canal oficial.
